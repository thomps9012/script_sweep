Request Headers

Authorization Headers​

Must be included on all requests outside of the /api/auth endpoint

email:​

Email of the person making the request

token:​

JWT retrieved from the /api/auth/jwt endpoint

Filters​

The options below can be specified in the request headers, and can be combined to create limitless, custom filtered search results

They are specific to GET requests on the scripts route
/api/scripts
/api/scripts/:id
/api/scripts/filter​

alive:​

Either true (still spoken) or false (not spoken)

// For example this request would return all non-living languages
const response = await fetch("/api/scripts", {
  method: "GET",
  headers: {
    alive: false,
  },
}).then((res) => res.json());

min_year:​

Any whole number / integer

Positive numbers are AD and negative numbers are BC

// Would return all languages that originated after the year 1090 AD
const response = await fetch("/api/scripts", {
  method: "GET",
  headers: {
    min_year: 1090,
  },
}).then((res) => res.json());

max_year:​

Any whole number / integer

Positive numbers are AD and negative numbers are BC

// Would return all languages that originated before the year 500 BC
const response = await fetch("/api/scripts", {
  method: "GET",
  headers: {
    max_year: -500,
  },
}).then((res) => res.json());

direction:​

Any of the three text direction options

LEFT_TO_RIGHT, RIGHT_TO_LEFT, TOP_TO_BOTTOM

// The query below would only return languages that read left to right
const response = await fetch("/api/scripts", {
  method: "GET",
  headers: {
    direction: "LEFT_TO_RIGHT",
  },
}).then((res) => res.json());

continent:​

One of the six Continent Options

AF [Africa], AS [Asia], AU [Australia], EU [Europe], NA [North America], SA [South America]

// Would return languages that were found on the Asiatic continent
const response = await fetch("/api/scripts", {
  method: "GET",
  headers: {
    continent: "AS",
  },
}).then((res) => res.json());

fields:​

Correlates to Table Columns

id, name, direction, year, ranges, living, link, continents

Separated by commas, with or without spaces 

// Would return only the id's and name's for the search result
const response = await fetch("/api/scripts", {
  method: "GET",
  headers: {
    fields: "id, name",
  },
}).then((res) => res.json());

limit:​

Any whole number (integer)

Limits total results returned by the request

// Would return two total languages
const response = await fetch("/api/scripts", {
  method: "GET",
  headers: {
    limit: 2,
  },
}).then((res) => res.json());

order_by:​

Field or Column to organize results by

id, name, continent, living, year, ranges, direction

// Organizes results by ascending id
const response = await fetch("/api/scripts", {
  method: "GET",
  headers: {
    order_by: "id",
  },
}).then((res) => res.json());

order_direction:​

Direction to organize results

Default is ascending, but reversed oder can be specified using the desc keyword

// Organizes results by descending id
const response = await fetch("/api/scripts", {
  method: "GET",
  headers: {
    order_by: "id",
    order_direction: "desc",
  },
}).then((res) => res.json());

These headers can be combined to receive custom, filtered search results

If you reach a 404 error, remove some filters to cast a broader search net
